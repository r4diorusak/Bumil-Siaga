## Open Source Dependencies

Aplikasi ini menggunakan teknologi dan library open source yang luar biasa:

### Frontend Frameworks & Libraries
- **Bootstrap** - Responsive CSS Framework
- **jQuery** - JavaScript Library
- **DataTables** - Advanced Table Display
- **Select2** - Enhanced Select Element
- **Summernote** - WYSIWYG Editor
- **SweetAlert2** - Beautiful Alert Dialogs
- **Toastr** - Notification Library

### Backend & Database
- **PHP PDO** - PHP Data Objects
- **MySQL/MariaDB** - Database Management System

### Icon & Font Libraries
- **Font Awesome** - Icon Library
- **Ionicons** - Icon Set
- **Raphael** - JavaScript Vector Graphics

### UI Components
- **Bootstrap Datepicker** - Date Selection
- **Bootstrap Colorpicker** - Color Selection
- **Bootstrap Slider** - Range Slider
- **Ion Range Slider** - Advanced Range Slider
- **Moment.js** - Date Manipulation
- **Tempusdominus Bootstrap** - DateTime Picker

### Chart & Visualization
- **Chart.js** - JavaScript Charting
- **Flot** - JavaScript Plotting Library
- **Morris.js** - Clean Charts
- **Sparklines** - Inline Charts

### Development Tools
- **Mocha** - Testing Framework
- **JSZip** - ZIP File Creation
- **PDFMake** - PDF Generation
